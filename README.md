# Desafío Pruebas Unitarias

## Descripción
Este desafío consiste la implementación de pruebas unitarias en una aplicación web con Vue JS.
Pruebas sobre rutas y snapshots para validar la estructura HTML

### Dependencias
- Vue CLI
- Jest
- Vue Test Utils
- Vue Router
